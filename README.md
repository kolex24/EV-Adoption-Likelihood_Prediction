# 🚗 EV Adoption Likelihood Prediction
> An end-to-end Machine Learning application that predicts the likelihood of electric vehicle (EV) adoption using demographic, behavioral, socio-economic and transportation-related features.
---
## Overview
Electric Vehicle (EV) adoption is accelerating worldwide as governments and autombile manufacturers promote sustainable transportation. However, understanding **who is likely to adopt an EV** remains a complex challenge. This project leverages Machine Learning to predict an individual's likelihood of adopting an electric vehhicle (EV) based on demograhic information, travel behavior, charging accessibility, environmental awareness, and financial characteristics.
The solution includes an interactive **streamlit application** that allow users to enter customer information and instantly receive and EV adoption prediction with probability score.
